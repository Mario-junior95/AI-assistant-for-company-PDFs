# AI Assistant for Company PDFs

Next.js app for uploading company PDFs and chatting with an AI grounded in those documents.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

| Route     | Description                                      |
| --------- | ------------------------------------------------ |
| `/`       | Chat — ChatGPT-style interface (UI ready)        |
| `/upload` | Upload — drag & drop PDFs (UI ready)             |

Built with [shadcn/ui](https://ui.shadcn.com) sidebar, scroll area, and form components.

## Scripts

| Command        | Description              |
| -------------- | ------------------------ |
| `npm run dev`  | Start development server |
| `npm run build`| Production build         |
| `npm run start`| Run production server    |
| `npm run lint` | Run ESLint               |
